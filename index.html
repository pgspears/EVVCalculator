Three ways to fill in a row

Each row in the table is one billable service. Use the Mode column to choose how that row is calculated:

Auto — Enter Start Time and End Time. Billable units and amount are computed using the 8-minute rule.
Best for: documenting completed visits.
Manual — Type the Billable Units directly. Times are optional and shown for reference only.
Best for: corrections, after-the-fact entry, or overriding the auto calculation.
Plan — Enter Start Time and Billable Units. The End Time is calculated for you.
Best for: scheduling shifts or working out when a visit needs to end to bill the desired number of units.
The 8-minute rule

Medicaid waiver services bill in 15-minute units. The first unit requires at least 8 minutes of service. After that, each additional 15 minutes earns one more unit.

Formula (when minutes ≥ 8): Units = floor((minutes − 8) ÷ 15) + 1

Times that span midnight (e.g., 11:00 PM → 1:00 AM) are handled automatically.

Daily cap: a maximum of 64 units (16 hours) can be billed per service per day. The calculator will display a prominent warning if any row exceeds this limit.
